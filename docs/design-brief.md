# Design Brief

## Product

Kikita UI is a general-purpose Angular 22+ UI library and design system.

It should work for admin tools, SaaS apps, dashboards, docs sites, content apps, and custom product interfaces. It must not be Discord-specific.

## Feel

- modern
- precise
- warm
- polished
- slightly animated
- production-grade
- not corporate-gray by default
- not noisy or childish

## Avoid

Do not copy the visual identity or API style of:

- Taiga UI
- Angular Material
- Ant Design
- PrimeNG
- Bootstrap
- shadcn
- Tailwind UI
- Radix themes

## Required Generated Boards

AI design tools should generate component boards, not only landing pages.

Required components for first boards:

- Button
- IconButton
- Icon
- Field
- Input
- Group
- Select
- Checkbox
- Switch
- Dialog
- Toast
- Card
- Tabs

Each board must include:

- light theme
- dark theme
- desktop layout
- mobile layout
- default state
- hover state
- active state
- focus state
- disabled state
- loading state where relevant
- error state where relevant
- success state where relevant

## Token Seeds

Visual directions must propose seed tokens for:

- primary
- neutral
- success
- warning
- danger
- radius
- density
- typography
- motion
