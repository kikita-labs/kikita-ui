All Kikita UI components adapt via data-kui-theme="dark" or "light" on any ancestor. Switching is instant. Nesting is planned for Phase 4. In the current version, nested accordion is not supported.
Three variants: default (dividers), bordered (individual blocks), ghost (no borders). Choose by context. Three sizes: sm 36px, md 44px (default), lg 52px. Set via size input on the container. In exclusive mode only one section is open. In multi mode the user can open any number simultaneously.
Accordion works well for FAQs, documentation, and any text-heavy blocks. Content is visible only when the user explicitly expands a section, reducing cognitive load and allowing more information without overwhelming the screen.
