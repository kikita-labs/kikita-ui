# Kikita UI

Angular 22+ UI library and design system.

Kikita UI is built for modern Angular applications using signals, Signal Forms, CSS variables, generated design tokens, and provider-based defaults.

## Status

Private experimental package.

Package name:

```text
@kikita-labs/ui
```

Future public package may become:

```text
@kikita/ui
```

## Local Development

Use Node 24.17.0+.

```bash
pnpm install
pnpm build
pnpm start
```

## Projects

```text
projects/ui         library package
projects/playground internal playground app
```

## Design Direction

The visual direction is not final yet. AI-generated component boards will be used to choose the first style direction, then tokens and components become the source of truth.
